# Jenkins Freestyle Project

## Introduction

This project demonstrates the basic functionality of Jenkins Freestyle Projects.

The project executes Linux shell commands from Jenkins to understand how build automation works.

---

## Workflow

```
Developer

↓

Build Now

↓

Jenkins Freestyle Job

↓

Workspace Created

↓

Execute Shell Script

↓

Console Output

↓

SUCCESS
```
